## Dinosaur game

Este é um projeto para a DIO com o objetivo de desenvolver um jogo bem simples inspirado no T-Rex  que aparece em uma página de erro no Chrome, ele foi desenvolvido utiizando HTML5, CSS3 e JavaScript.
